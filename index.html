I Design and Develop Unique Web Experiences

Here's the Process...

Design

In order to design a site, we must first start with the message. A complete understanding of what you are trying to say leads right into color schemes, lines and design. This stage can be as simple as sketches on paper and wireframes to a full blown "pixel perfect" rendered image in photoshop. Your call.

Develop

The code base takes takes the design to as many screens as possible. The code must be as pristine as the design, well-commented with liberal use of whitespace throughout. All code is accompanied by a detailed README file describing the code base so that I or anyone else can understand what is going on under the hood.

Deliver

When the code is written and the site is working, then the testing begins. Some say it never stops. Once the code base has passed a rigorous testing process then we deliver the files to the server. There the website lies until such a time as it needs to be updated and the process begins again.
